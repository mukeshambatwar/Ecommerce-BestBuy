---------------------------------------------------------------

* Creating Virtual Environment 
-------------------------------
  - Create the folder-> open that folder with GitBash type following command -> 
    $python -m venv env
  - To activate virtual Environment you have to ender following command
    $source env/Scripts/activate
  - To deactivate or comeout from the env 
    $deactivate
